Resistance Remastered Controls Patch

Put 'resistance_remastered.prx' at 'ux0:pspemu/seplugins/resistance_remastered.prx' and write this line to 'ux0:pspemu/seplugins/game.txt': 'ms0:/seplugins/resistance_remastered.prx 1' (Without ')

If you create a file called 'ux0:pspemu/seplugins/resistance_infected.bin', the infected mode can be activated.

R2/L2 (weapon select) is mapped to the right/left arrows.
R3 (aim) is mapped to the down arrow.
